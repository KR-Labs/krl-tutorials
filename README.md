## Learning Paths

### Beginner: Foundation & Exploration
Start your analytics journey by mastering the fundamentals. Learn to connect, load, and explore datasets from trusted sources including the Federal Reserve, U.S. Census Bureau, and World Bank. Build your first visualizations and develop confidence in working with economic and social data.

### Intermediate: Integration & Analysis
Deepen your expertise by combining multiple data sources, conducting time series analysis, and creating compelling geographic visualizations. Master the techniques that transform raw data into meaningful insights for strategic decision-making.

### Advanced: Causal Inference & Impact Evaluation
Apply sophisticated methodologies to evaluate real-world scenarios. Learn difference-in-differences, synthetic control methods, and regression discontinuity designs to rigorously assess policy impacts and economic interventions.

### Expert: Production & Deployment
Transition from analysis to implementation. Build interactive dashboards, deploy machine learning models, develop data APIs, and optimize performance for enterprise-scale applications.

## Getting Started

**1. Access the Tutorials**  
Clone the repository and open any notebook in Jupyter Lab or Jupyter Notebook to begin your learning journey.

```bash
git clone https://github.com/KR-Labs/krl-tutorials.git
cd krl-tutorials
pip install -r requirements.txt jupyter jupyterlab
jupyter lab
```

**2. Choose Your Path**  
Start with beginner fundamentals or dive directly into topics relevant to your work. Each tutorial is self-contained with clear learning objectives and practical exercises.

**3. Learn by Doing**  
Every tutorial combines conceptual explanation with hands-on code examples. Practice with real datasets, complete exercises, and build skills progressively.

**4. Engage with the Community**  
Share insights, ask questions, and contribute improvements. Join discussions at [github.com/orgs/KR-Labs/discussions](https://github.com/orgs/KR-Labs/discussions).

### What You Need

- **Basic Python knowledge** (variables, functions, data structures)
- **Statistical familiarity** (descriptive statistics, basic probability)
- **API keys** for data sources (free registration with FRED, Census Bureau, BLS)

Complete setup instructions and data source guides are provided within the tutorials.
